Every element is able to have:

Smooth scroll =>
	1. Appearance / Menus / Custom Links => URL : #name_of_destination, Link Text : link_text
	2. Sections / some section / Custom fields / Name : smooth_scroll_id , Value : some ID

Add new Section =>
	1. Go to Sections in Dashboard
	2. Add new section

Add custom class for background color =>
	1. Sections / some section / Custom fields / Name : wrapper_class , Values : [ gray-bg, blue-bg, green-bg, padding-2em ] or you can set [ border-bottom-gray ]

Section can be without title:
	1. Custom Fields of any section :
		1.1. section_without_title
		1.2. section_without_title_button_color
		1.3. section_without_title_button_target
		1.4. section_without_title_button_text
